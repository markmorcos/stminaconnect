# Release Notes — v1.0.0

Initial public release. Targets Play Console's per-locale "What's new" field (500-char limit per locale). Paste the locale blocks verbatim into the corresponding fields when promoting a release in Play Console → Production → Create new release → Release details → Release notes.

The same blocks work in App Store Connect's per-locale "What's New in This Version" field.

> AR and DE drafts SHOULD be reviewed by a native speaker before submission, same posture as `docs/store/listings/`.

## Play Console / App Store Connect blocks (paste-ready)

```
<de-DE>
Willkommen bei St. Mina Connect — ein Begleiter für die Seelsorge in Kirchengemeinden.

Diese erste Version bietet Quick Add zum schnellen Erfassen von Mitgliedern, Gruppen-Anwesenheit für Gottesdienste und Treffen, Nachbetreuung für fehlende Mitglieder und ein Mitarbeiter-Dashboard. Verfügbar auf Deutsch, Englisch und Arabisch.

Datenschutz zuerst: Daten in der EU, keine Werbung, kein Tracking. DSGVO-Rechte (Zugriff, Export, Löschung) sind eingebaut.
</de-DE>
```
